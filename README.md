# 🖧 School IT Infrastructure & Network + CCTV Management System

## 📌 Overview
This project documents the complete IT infrastructure management of a private school environment, including network administration, centralized internet management, Google Workspace administration, system troubleshooting, and IP-based CCTV surveillance integration.

It reflects real-world IT support and network administration tasks performed in a school setting with a focus on stability, security, and centralized control.

---

## 🏫 Environment Summary
- Type: Private Educational Institution
- Users: Students, Teachers, Administrative Staff
- Systems Managed:
  - Network Infrastructure (LAN/Wi-Fi)
  - Google Workspace Administration
  - Enrollment System Support
  - IP CCTV Surveillance System
  - IT Support Services

---

# 🌐 NETWORK INFRASTRUCTURE (TP-LINK OMADA CONTROLLER)

## 📌 Overview
The school network is centrally managed using the TP-Link Omada SDN system with Omada Controller, allowing unified control of all network devices including routers, switches, and access points.

---

## 🧠 Core System Components
- Omada Controller (Software / Hardware / Cloud-Based Management)
- Omada Gateway Router
- Managed PoE Switches
- Omada Wireless Access Points

---

## 🏫 Network Architecture
- Centralized management through Omada Controller
- All devices linked under one SDN system
- LAN and Wi-Fi fully controlled via dashboard
- Real-time monitoring of network performance

---

## 📡 Network Segmentation (VLAN Structure)
- Admin Network – Secure access for school administration
- Faculty Network – Staff and teacher devices
- Student Network – Controlled access and bandwidth limits
- CCTV Network – Isolated IP camera system for security

---

## ⚙️ Omada Controller Responsibilities
- Centralized device management (router, switch, APs)
- VLAN and SSID configuration
- Wireless network optimization
- Real-time traffic monitoring and analytics
- Remote troubleshooting and device rebooting
- Client access control and bandwidth management

---

## 🔐 Security Configuration
- WPA2/WPA3 Wi-Fi encryption
- VLAN isolation between departments
- Guest network restrictions
- Access control rules via Omada Controller
- Device-level monitoring and blocking if needed

---

## 📈 Benefits
- Centralized control of entire school network
- Easier troubleshooting via dashboard
- Improved Wi-Fi stability and coverage
- Faster issue detection and resolution
- Reduced downtime with real-time monitoring

# 📹 IP CCTV SURVEILLANCE SYSTEM

## 📌 Overview
Installed and managed IP-based CCTV system integrated into the school network.

---

## 🧰 Components
- IP Cameras (PoE / Wi-Fi)
- Network Switch (PoE)
- NVR (Network Video Recorder)
- LAN infrastructure

---

## 🌐 Setup
- Cameras assigned static IP addresses
- Connected to centralized switch/router
- NVR configured within same network
- Live monitoring via admin workstation

---

## 🗺️ Camera Locations
- Entrance Gate
- Hallways
- Computer Laboratory
- Administration Office
- School Perimeter

---

## 🔐 Security
- Restricted admin access
- Strong password protection
- VLAN isolation for CCTV network

---

## ⚙️ Troubleshooting
- Camera not detected → IP conflict or cable issue
- Video lag → bandwidth limitation
- NVR access error → wrong subnet/IP

---

# 🔧 GENERAL TROUBLESHOOTING
